- # Termux qurulum

1. Reponu klonla
   ```markdown
   pkg install git && git clone https://github.com/Guliyev-di/Multi-Termux
   ```
   
2. Repoya daxil ol
    ```markdown
    cd Multi-Termux
   ```

3. Lazımlı faylları yüklə
    ```markdown
   pkg update && pkg upgrade && pkg install python && pkg install python-pip && pip3 install -U pip && pip3 install colorama

    ```

4. Kodu başlad
    ```markdown
   python3 script.py
    ```
